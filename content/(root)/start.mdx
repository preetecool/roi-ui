## Installation

### 1. Install Base UI
Roi UI components are built on top of Base UI. You'll need to install Base UI first to use these components.

<CodeBlockTabs noStyleSuffix>

<CodeBlockTabsList>
  <CodeBlockTabsTrigger value="npm">npm</CodeBlockTabsTrigger>
  <CodeBlockTabsTrigger value="pnpm">pnpm</CodeBlockTabsTrigger>
</CodeBlockTabsList>

<CodeBlockTab value="npm">

```bash
npm install @base-ui/react
```

</CodeBlockTab>

<CodeBlockTab value="pnpm">

```bash
pnpm add @base-ui/react
```

</CodeBlockTab>

</CodeBlockTabs>

### 2. Install utils
 The tailwind utils file will not have the -tailwind suffix when it's in your registry.

<CodeBlockTabs>

<CodeBlockTabsList>
  <CodeBlockTabsTrigger value="npm">npm</CodeBlockTabsTrigger>
  <CodeBlockTabsTrigger value="pnpm">pnpm</CodeBlockTabsTrigger>
</CodeBlockTabsList>

<CodeBlockTab value="npm">

```bash if=css-modules
npx shadcn@latest add @roiui/utils
```

```bash if=tailwind
npx shadcn@latest add @roiui/utils-tailwind
```

</CodeBlockTab>

<CodeBlockTab value="pnpm">

```bash if=css-modules
pnpm dlx shadcn@latest add @roiui/utils
```

```bash if=tailwind
pnpm dlx shadcn@latest add @roiui/utils-tailwind
```

</CodeBlockTab>

</CodeBlockTabs>



### 3. Set up portals

Add a portal container to your root layout:

<CodeBlock
  filename="layout.tsx"
  language="tsx"
  code={`<body>
  <div className="root">
    {children}
  </div>
</body>`}
/>
